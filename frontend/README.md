# Retail Cortex Frontend

React frontend for Retail Cortex platform.

## Setup

```bash
npm install
```

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Environment Variables

Create a `.env` file:

```
VITE_API_URL=http://localhost:8000/api
```

For production, set this to your deployed API URL.
